# RCB IPL 2025 Analytics Portfolio

A single-page portfolio built from an Excel analysis workbook covering:

- RCB final match batting and bowling dashboards
- Ball-by-ball score progression
- Group-stage batting phase analysis
- Group-stage bowling phase analysis
- Pivot-table opponent insights
- Power Pivot, XLOOKUP, and dashboarding workflow

Open `index.html` locally or publish the repository with GitHub Pages.
